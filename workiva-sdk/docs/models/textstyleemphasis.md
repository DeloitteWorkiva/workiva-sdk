# TextStyleEmphasis

The emphasis formats applied by the text style. This consists of the bold, italic, and underline formats.



## Fields

| Field                                 | Type                                  | Required                              | Description                           | Example                               |
| ------------------------------------- | ------------------------------------- | ------------------------------------- | ------------------------------------- | ------------------------------------- |
| `bold`                                | *Optional[bool]*                      | :heavy_minus_sign:                    | Whether or not the text is bold       | false                                 |
| `italic`                              | *Optional[bool]*                      | :heavy_minus_sign:                    | Whether or not the text is italicized | false                                 |
| `locked`                              | *Optional[bool]*                      | :heavy_minus_sign:                    | Whether the emphasis group is locked. | false                                 |
| `underline`                           | *Optional[bool]*                      | :heavy_minus_sign:                    | Whether or not the text is underlined | false                                 |
