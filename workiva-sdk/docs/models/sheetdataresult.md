# SheetDataResult

Returns a JSON object with `data` and `@nextLink` properties. `data` contains a  [`SheetData`](ref:spreadsheets#sheetdata) object, and `@nextLink`  provides the URL to the next set of results. If there are no additional results,  `@nextLink` doesn't appear.
