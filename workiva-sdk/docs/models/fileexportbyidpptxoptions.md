# FileExportByIDPptxOptions

Optional options to export the presentation as a Microsoft Powerpoint File (.pptx). If no options are provided, all options default to False.

| ---------------------------------------------------------- | ---------------------------------------------------------- | ---------------------------------------------------------- | ---------------------------------------------------------- | ---------------------------------------------------------- |
| `include_unused_layout_slides`                             | *Optional[bool]*                                           | :heavy_minus_sign:                                         | Whether to include unused layout slides. False by default. | true                                                       |
