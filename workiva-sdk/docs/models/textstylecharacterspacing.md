# TextStyleCharacterSpacing

The character spacing applied by the text style. Setting this format requires the Edit Character Spacing ability.


## Fields

| Field                                             | Type                                              | Required                                          | Description                                       | Example                                           |
| ------------------------------------------------- | ------------------------------------------------- | ------------------------------------------------- | ------------------------------------------------- | ------------------------------------------------- |
| `character_spacing`                               | *Optional[float]*                                 | :heavy_minus_sign:                                | The space between characters, in 1/1000 of an em. | -25                                               |
| `locked`                                          | *Optional[bool]*                                  | :heavy_minus_sign:                                | Whether the space between characters is locked.   | false                                             |
