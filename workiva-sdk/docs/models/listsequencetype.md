# ListSequenceType

The type of list sequence. If the list style is unordered, the sequence type of the first list progression must be unordered. If the list style is ordered, the sequence type of the first list progression may not be unordered.



## Values

| Name                            | Value                           |
| ------------------------------- | ------------------------------- |
| `UNORDERED_SEQUENCE`            | unorderedSequence               |
| `NUMERIC_SEQUENCE`              | numericSequence                 |
| `UPPERCASE_ALPHABETIC_SEQUENCE` | uppercaseAlphabeticSequence     |
| `LOWERCASE_ALPHABETIC_SEQUENCE` | lowercaseAlphabeticSequence     |
| `UPPERCASE_ROMAN_SEQUENCE`      | uppercaseRomanSequence          |
| `LOWERCASE_ROMAN_SEQUENCE`      | lowercaseRomanSequence          |
