# RichTextDuplicationEditListResult

Returns a JSON object with `data` and `@nextLink` properties. `data` contains a list of [`RichTextDuplicationEditResult`](ref:content#richtextduplicationeditresult) objects, and `@nextLink` provides the URL to the next set of results. If there are no additional results, `@nextLink` doesn't appear. If the request returns no results at all, `data` contains an empty array.
