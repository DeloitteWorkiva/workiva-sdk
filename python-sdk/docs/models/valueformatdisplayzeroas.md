# ValueFormatDisplayZeroAs

The word to use for zero. Valid for ACCOUNTING, CURRENCY, NUMBER, and PERCENT.


## Values

| Name         | Value        |
| ------------ | ------------ |
| `ZERO`       | ZERO         |
| `NO`         | NO           |
| `NONE`       | NONE         |
| `NOTHING`    | NOTHING      |
| `NIL`        | NIL          |
| `NOT`        | NOT          |
| `NOMINAL`    | NOMINAL      |
| `IMMATERIAL` | IMMATERIAL   |
