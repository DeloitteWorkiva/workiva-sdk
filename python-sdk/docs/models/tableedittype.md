# TableEditType

The type of edit operation to perform on the table


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `DELETE_COLUMNS` | deleteColumns    |
| `DELETE_ROWS`    | deleteRows       |
| `MOVE_COLUMNS`   | moveColumns      |
| `MOVE_ROWS`      | moveRows         |
| `INSERT_COLUMNS` | insertColumns    |
| `INSERT_ROWS`    | insertRows       |
