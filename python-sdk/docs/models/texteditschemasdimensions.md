# TextEditSchemasDimensions

The dimensions of the inserted table. If a width or height are omitted the width or height will be defaulted based off the number of columns or rows.



## Fields

| Field                                 | Type                                  | Required                              | Description                           | Example                               |
| ------------------------------------- | ------------------------------------- | ------------------------------------- | ------------------------------------- | ------------------------------------- |
| `height`                              | *OptionalNullable[float]*             | :heavy_minus_sign:                    | The vertical measurement in points.   | 240                                   |
| `width`                               | *OptionalNullable[float]*             | :heavy_minus_sign:                    | The horizontal measurement in points. | 320                                   |
