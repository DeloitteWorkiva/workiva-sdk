# StrokePattern

The pattern to apply to the stroke.


## Values

| Name                | Value               |
| ------------------- | ------------------- |
| `SOLID`             | solid               |
| `DASHED`            | dashed              |
| `DOTTED`            | dotted              |
| `DASH_DOT`          | dash-dot            |
| `DITTED`            | ditted              |
| `SMALL_DASH`        | small-dash          |
| `LONG_DASH`         | long-dash           |
| `LONG_DASH_DOT`     | long-dash-dot       |
| `LONG_DASH_DOT_DOT` | long-dash-dot-dot   |
