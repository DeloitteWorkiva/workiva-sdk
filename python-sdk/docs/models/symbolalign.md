# SymbolAlign

Where to render the symbol relative to the value. All values valid for ACCOUNTING and CURRENCY. Left values valid for NUMBER. Right values valid for PERCENT.


## Values

| Name                  | Value                 |
| --------------------- | --------------------- |
| `SYMBOL_DEFAULT`      | SYMBOL DEFAULT        |
| `LEFT`                | LEFT                  |
| `LEFT_INSIDE`         | LEFT INSIDE           |
| `RIGHT`               | RIGHT                 |
| `RIGHT_INSIDE`        | RIGHT INSIDE          |
| `RIGHT_SPACED_INSIDE` | RIGHT SPACED INSIDE   |
| `RIGHT_SPACED`        | RIGHT SPACED          |
