# Dimensions

A dimension of a rectangle for properties of objects such as a shape, text box, table, chart, image, or page.



## Fields

| Field                                 | Type                                  | Required                              | Description                           | Example                               |
| ------------------------------------- | ------------------------------------- | ------------------------------------- | ------------------------------------- | ------------------------------------- |
| `height`                              | *float*                               | :heavy_check_mark:                    | The vertical measurement in points.   | 240                                   |
| `width`                               | *float*                               | :heavy_check_mark:                    | The horizontal measurement in points. | 320                                   |
