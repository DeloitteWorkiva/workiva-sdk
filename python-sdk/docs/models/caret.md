# Caret

A caret (as in Caret Navigation) is a location between two characters in a paragraph.


## Fields

| Field                                                                               | Type                                                                                | Required                                                                            | Description                                                                         | Example                                                                             |
| ----------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| `offset`                                                                            | *int*                                                                               | :heavy_check_mark:                                                                  | The number of characters to skip over in a paragraph to where the Caret is located. | 10                                                                                  |
| `paragraph_index`                                                                   | *int*                                                                               | :heavy_check_mark:                                                                  | The zero-based index of a paragraph.                                                | 1                                                                                   |
