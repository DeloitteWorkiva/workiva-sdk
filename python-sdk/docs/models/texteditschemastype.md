# TextEditSchemasType

The type of hyperlink to create. The property associated with this hyperlink type is required.


## Values

| Name           | Value          |
| -------------- | -------------- |
| `DOCUMENT`     | document       |
| `SECTION`      | section        |
| `PRESENTATION` | presentation   |
| `SLIDE`        | slide          |
| `SPREADSHEET`  | spreadsheet    |
| `SHEET`        | sheet          |
| `URL`          | url            |
